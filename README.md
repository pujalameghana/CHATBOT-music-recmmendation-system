# Chatbot-Based Music Recommendation System 🎵🤖

## Description
The Chatbot-Based Music Recommendation System is an intelligent application that recommends songs to users based on their mood and preferences through a conversational chatbot interface. The system uses Machine Learning and Natural Language Processing (NLP) techniques to understand user input and provide relevant song suggestions.

The chatbot interacts with users through text conversations, analyzes the input using NLP techniques, and recommends songs based on detected mood or keywords. This project demonstrates how conversational AI can enhance the music discovery experience.

---

## Features
- Chatbot-based interaction with users
- Mood-based song recommendations
- User registration and login system
- Admin panel for managing songs
- Song playback functionality

---

## Technologies Used
- Python
- Machine Learning
- Natural Language Processing (NLP)
- NumPy
- Pandas
- Scikit-learn
- TensorFlow

---

## System Workflow
1. User registers or logs into the system.
2. User interacts with the chatbot.
3. The chatbot processes the user's message using NLP.
4. The system identifies the user's mood or preferences.
5. Based on the analysis, the system recommends suitable songs.
